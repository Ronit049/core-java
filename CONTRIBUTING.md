🤝 Contributing to Core Java Repository

Thank you for your interest in contributing! 🚀
This repository is aimed at helping beginners and students learn Core Java effectively.
---
📌 How to Contribute

Follow these steps to contribute:

Fork the Repository
Click on the "Fork" button on the top right
Clone Your Fork
```
git clone https://github.com/Ronit049/core-java.git
```
Create a Pull Request
Go to your forked repo
Click on "New Pull Request"
Describe your changes clearly
---
📂 Contribution Ideas

You can contribute in the following ways:

✅ Add basic Java programs (loops, arrays, strings)
✅ Implement OOP concepts (classes, inheritance, polymorphism)
✅ Add exception handling examples
✅ Add file handling programs
✅ Improve existing code
✅ Fix bugs
✅ Improve documentation
---
🧑‍💻 Coding Guidelines
Use meaningful variable names
Follow proper Java naming conventions
Add comments to explain your code
Keep code clean and readable
Avoid unnecessary complexity
---
📁 Folder Structure

Try to organize code like this:
```
Core-Java/
│── Basics/
│── OOP/
│── Collections/
│── Exception-Handling/
│── File-Handling/
```
📜 Commit Message Guidelines

Use clear and simple commit messages:

Add: New Java program for loops
Fix: Bug in array sorting code
Update: Improved README
---
🚫 Things to Avoid
❌ Do not submit duplicate programs
❌ Do not add unnecessary files
❌ Do not write messy or unstructured code
---
💬 Need Help?

If you have any questions, feel free to open an issue.
